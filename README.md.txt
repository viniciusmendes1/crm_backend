# CRM para Revista – Aplicação Windows

## Descrição

Este projeto é um CRM desenvolvido em Python para ser instalado no Windows, com a finalidade de atuar como um CRM para uma revista. O programa possui uma interface gráfica amigável e permite a gestão de releases, contatos de assessorias e mailing de imprensa.

## Tecnologias Utilizadas

- Python
- PyQt5 (para GUI)
- SQLite (para banco de dados)
- Pandas (para manipulação de dados)
- IMAP (para leitura de e-mails)
- PyInstaller (para criar o instalador Windows)

## Funcionalidades

- Interface Gráfica amigável
- Módulo de Releases
- Gestão de Contatos
- Mailing de Imprensa
- Integração com E-mail
- Sistema de notificações
- Interface com modo escuro e claro

## Estrutura do Projeto

- `main.py` - Arquivo principal para iniciar a aplicação
- `gui/` - Diretório contendo os arquivos da interface gráfica
- `modules/` - Diretório contendo os módulos de releases, contatos e mailing
- `database/` - Diretório contendo o banco de dados SQLite
- `utils/` - Diretório contendo arquivos utilitários

## Instalação

Para instalar e executar o programa, siga as instruções abaixo:

1. Clone este repositório.
2. Instale as dependências usando `pip install -r requirements.txt`.
3. Execute o arquivo `main.py`.
